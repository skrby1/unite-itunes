# unite-itunes
Mac用のMusicアプリのプレイリストやトラックをリスト表示して再生するUniteソースです。  
便宜上unite-itunesと名前はついていますが、現在はCatalinaにしか対応していません。  

Requirements
-----
- [Unite](https://github.com/Shougo/unite.vim)

使い方
-----

    :Unite itunes
    :Unite itunes:!
    :Unite itunes:a:アルバム名
    :Unite itunes:n:アーティスト名
    :Unite itunes:y:曲の年
    :Unite itunes:t:曲のタイトル
    :Unite itunes:[<>]:最小分数-最大分数
